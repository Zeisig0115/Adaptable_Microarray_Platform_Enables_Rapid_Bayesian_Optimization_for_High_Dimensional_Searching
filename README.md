# Adaptable-Microarray-Platform-Enables-Rapid-Bayesian-Optimization-for-High-Dimensional-Searching
This is a repo for using Bayesian Optimization to search the optimal essential + additive concentration combinations with mixed inputs, NChooseK and continuous inputs. Using BoTorch APIs.
